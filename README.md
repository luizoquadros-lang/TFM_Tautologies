# The Meaning of Tautologies: Experimental Analysis

This repository contains codes and dataset related to the Master’s thesis:

**“The Meaning of Tautologies: An Experimental Study of their Pragmatic Interpretation"** (Quadros, Luiz. 2026)

## Project Overview

This project investigates how equative tautological constructions (e.g. *“boys are boys”*, *“a promise is a promise”*) are interpreted by speakers, following the semantic-pragmatic classification proposed by Wierzbicka (1987).

The study combines:
- quantitative analysis (Likert-scale ratings)
- qualitative analysis (open-ended responses)

to examine how structure influences interpretation across different categories:
- inevitability
- tolerance
- obligation

## Files included in the repository

### Datasets

All data was collected using Google Forms and processed in R.
The datasets included here correspond to:
- dataset_TFM_final: Final cleaned version of all **39 participants** responses combined from datasets Version A and B. 
- version_A_original: Raw answers from google-forms-exported version dataset containing 20 responses.
- version_B_original: Raw answers from google-forms-exported version dataset containing 19 responses.

### Code

The analysis was conducted using **R**, including:

- TFM_dataset_cleaning_preprocessing: Script used for the cleaning process.
- TFM_main_analysis: Script with the analysis, calculation of mean ratings.
